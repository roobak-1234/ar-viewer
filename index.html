Multi-User AR App

A deep-dive into the architectural synchronization of collaborative augmented reality environments.

Executive Summary

This repository documents the design and development phases of a state-of-the-art multi-user AR heart simulation. By integrating peer-to-peer networking at the transformation layer, we achieve sub-millisecond interaction parity between iOS and Android clients.

Network Jitter Buffer

↓
When users are on different Wi-Fi networks, packet arrival is inconsistent. We solved this by implementing a Lerp-based prediction algorithm that smooths out transformations between network update frames.

Transform Race Conditions

↓
Two users grabbing the same part simultaneously. We implemented a "State Ownership" ID system where the first interactor takes exclusive control of the networking layer for that object until released.
